# Anedya-Example-ESP_IDF

## Menu-Config Setup
    1. Uncheck
        - Store phy calibration data in NVS
        - WiFi NVS flash
    2. Select Flash Size 4MB.
    3. Select `Partition Table` -> Custome partition table csv
    4. Fill Anedya Credentials
        - Connection Key
        - Physical Device ID
    5. Fill Wifi Credentials
        - SSID
        - Password
